# ShallotT v1.0.8 — Release Notes

## Extension navigateur (Firefox + Chrome)

### Nouvelles fonctionnalités

| Fonctionnalité | Raccourci | Détail |
|---|---|---|
| 🌐 **Traduction de page entière** | `Ctrl+Shift+F10` | Traduit tout le texte visible en conservant le layout. Toggle pour revenir à l'original. Cache : le 2e switch est instantané. |
| 📸 **OCR intégré** | `Ctrl+Shift+F8` | Tesseract.js WASM bundlé. Sélection d'une zone à l'écran → extraction → traduction. Zéro dépendance externe. |
| ⚡ **Quick translate** | `Ctrl+F9` + lettre | Appuie `Ctrl+F9` puis une lettre (E=English, F=Français, S=Español...). Mapping éditable dans les settings. |
| 🔍 **Auto-détection** | Souris | Bouton "🌐 Traduire en [langue] ?" qui apparaît automatiquement sous le texte sélectionné. |
| 💬 **Bulle inline** | `Alt+Shift+T` | Affiche la bulle de traduction flottante (comme le clic droit). |

### Corrections

| Bug | Status |
|---|---|
| Settings panel doublait la largeur de la popup | ✅ Fixé — overlay `position: absolute` |
| Impossible de scroller dans les settings | ✅ Fixé — `overflow-y: auto` |
| Resize de la bulle agrandissait les boutons | ✅ Fixé — `flex: 1` sur la zone texte uniquement |
| Clic droit changeait la langue définitivement | ✅ Fixé — one-shot `_contextLang` |
| Shortcut Manager ouvrait chrome:// sur Firefox | ✅ Fixé — `browser.commands.openShortcutSettings()` |
| Auto-detect : bulle ne s'ouvrait pas au clic | ✅ Fixé — `browser.runtime.sendMessage` + `querySelector` |
| Page translate : prompt affiché au lieu de la trad | ✅ Fixé — format `[N]` avec `num_predict: 4096` |
| Tesseract.js "not defined" | ✅ Fixé — injection via `files: []` en shared world |

### Raccourcis

| Raccourci | Action |
|---|---|
| `Ctrl+Shift+Y` | Ouvrir la popup |
| `Alt+Shift+T` | Traduire la sélection (bulle) |
| `Ctrl+F9` + lettre | Quick translate par langue |
| `Ctrl+Shift+F8` | OCR zone (Tesseract.js) |
| `Ctrl+Shift+F10` | Traduire toute la page (toggle) |

## Application desktop (Windows / macOS / Linux)

### Nouvelles fonctionnalités

| Fonctionnalité | Détail |
|---|---|
| 🎨 **16 thèmes** | Dark, Light, Nord, Solarized, Sépia, Contraste, Hacker, Dragon, Classic Mac, Retro Amber, Retro Green, Dracula, Synthwave, Monokai, Tokyo Night, NFS Tokyo Drift |
| 🖼️ **Wallpaper personnalisé** | JPG/PNG en fond, modes Tile/Center/Stretch/Fit, transparence réglable |
| 🐲 **Décorations overlay** | Dragon : tête + queue qui dépassent. NFS : 🏎️💨🔥. Windows/Mac uniquement. |
| 🎤 **Audio STT** | Capture micro + faster-whisper → traduction automatique |
| 📊 **Cache SQLite** | Translations stockées, rappel instantané |
| 📚 **Glossaire** | Termes à ne pas traduire, injectés dans le prompt |
| 🔊 **TTS** | Lecture vocale offline (pyttsx3) |
| 📋 **Clipboard Monitor** | Traduction popup automatique du presse-papier |
| 📄 **Drag & Drop** | Import .txt .pdf .srt .docx |
| 📜 **Historique** | 50 dernières traductions avec recherche |
| 📤📥 **Export/Import** | Sauvegarde/restauration de la config en JSON |
| 🔗 **OCR Bridge** | Serveur HTTP localhost:11435 pour l'extension |

### Build Windows

- `--onedir` par défaut (anti faux-positif antivirus)
- UPX désactivé
- Métadonnées Windows intégrées (version_info.txt)
